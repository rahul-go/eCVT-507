\subsection{Detailed Description}
Header for a library that implements the input/output configuration. 

This library allows the user to configure the inputs/outputs in one place. The library assigns ports and pin numbers via \mbox{\hyperlink{struct_pin}{Pin}} objects to each input/output. Additionally, the library sets up interrupts for the engine speed and rear wheel speed hall effect sensors.
